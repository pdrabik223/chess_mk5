# Micropython 

1. Install Thonny
2. Flash Board using Micropython utf2 file
3. Clone library 
4. Download nescessary submodules
5. Open thony and select you'r board 
6. Flash additional nescessary files to pi memory
7. Either run main.py file using Thonny
8. Or flash main.py file to board and it will run after board is connected to power
